# Environments
* node @16.0.0
(latest node version will crash with **react-scripts@4.0.3**)

# Installation
```
git clone <this repository>
yarn install
```
* place *.env* file at the root directory
* place *artifacts/* directory under the *src/* directory

```
npm start
```
